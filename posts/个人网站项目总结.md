# 个人网站项目总结

> 2026年7月14日 · 日常

## 项目概述

这是我的个人网站，托管在 GitHub Pages 上，采用纯静态方案构建。整个网站从零开始，借助 AI（Vibe Coding）完成开发，目前已经迭代了多个版本，功能趋于完善。这篇文章将详细介绍网站的各个功能模块。

---

## 技术栈

网站的技术选型贯彻"极简、够用、零依赖"的原则：

- **HTML5** — 页面结构，语义化标签
- **Tailwind CSS** — 通过 Play CDN 引入，直接在 class 中写样式，无需构建工具
- **原生 JavaScript** — 不依赖任何框架，所有交互逻辑手写
- **marked.js** — Markdown 渲染库，用于博客文章和项目详情的渲染
- **Font Awesome 6** — 图标库，通过 CDN 引入
- **GitHub Pages** — 免费托管，自动部署，push 即上线

不需要 `npm install`，不需要 `webpack` 配置，写完直接推送到 GitHub 就能上线，维护成本极低。

## 主页功能介绍

### 导航栏

固定在页面顶部的导航栏，采用毛玻璃效果（`backdrop-blur`），滚动时背景加深并添加阴影。导航包含"首页"、"关于我"、"常用链接"、"联系我"四个锚点链接，以及一个"博客"按钮（渐变紫色胶囊样式，跳转到 `blog.html`）。

移动端适配：在中小屏幕上自动隐藏横向导航，改为汉堡菜单按钮，点击展开垂直菜单，点击任意链接后自动收起。

**1. 中国古典诗词随机轮播**

渐变蓝紫分割线、背景渐变遮罩（从黑到蓝紫）、底部渐变淡出过渡，增强视觉层次。

## 博客功能介绍

博客系统是本网站的核心模块之一，包含三个页面：`blog.html`（列表页）、`post.html`（文章详情页）、`project.html`（项目详情页）。

### Blog 列表页 (`blog.html`)

**背景视频**

博客页使用动态视频背景（非静态渐变），提供 4 个可切换的视频源：
- BITTER-SPICE 动态壁纸
- AMY 4K
- Chill Cat Coffee
- bizhi

视频以 `fixed` + `object-fit: cover` 全屏铺满，上方覆盖半透明黑色遮罩。用户可通过导航栏的视频切换按钮（下拉菜单）更换背景，选择会保存到 `localStorage` 以便下次访问恢复。

**透明度调节**

在文章页和项目页的导航栏上，有一个调节按钮（齿轮图标），点击展开一个下拉面板，提供三个滑块：
- 遮罩透明度（0% ~ 90%）
- 卡片透明度（20% ~ 95%）
- 毛玻璃模糊（0px ~ 40px）

所有参数实时生效，并保存到 `localStorage`，每次打开页面自动恢复用户偏好。

**Tab 切换**

博客页顶部有两个 Tab：**文章** 和 **项目**。默认显示文章列表，点击切换后显示项目列表，两个 Tab 各有一套独立的筛选和渲染逻辑。

**文章列表功能：**
- **搜索**：顶部搜索框，实时过滤标题和摘要
- **分类筛选**：从 `posts.json` 动态提取所有标签（如"日常"、"汇总"），生成筛选按钮，点击切换
- **统计卡片**：总文章数、标签分类数、最近更新日期
- **文章卡片**：每篇文章显示封面图标（渐变背景）、标签、日期、标题、摘要，点击跳转到 `post.html?slug=xxx`

**项目列表功能：**
- **分类筛选**：从 `projects.json` 动态提取项目分类
- **统计卡片**：总项目数、已完成数、开发中数、项目分类数
- **项目卡片**：显示分类标签、状态（已完成/开发中）、技术标签、日期、标题、摘要，点击跳转到 `project.html?slug=xxx`

数据加载失败时显示错误提示，加载中显示骨架屏动画。

### 文章详情页 (`post.html`)

通过 URL 参数 `?slug=xxx` 定位文章：
1. 从 `posts.json` 获取文章元信息（标题等），动态设置 `<title>`
2. 从 `posts/{slug}.md` 获取 Markdown 正文
3. 使用 `marked.js` 将 Markdown 渲染为 HTML

页面支持与博客页相同的背景视频和透明度调节功能。导航栏提供"返回博客"和"返回首页"链接。

### 项目详情页 (`project.html`)

机制与文章详情页类似，但加载的是 `projects/` 目录下的 Markdown 文件，并有专门针对项目内容的 CSS 样式（代码块、标题层级等）。

## 数据驱动设计

整个网站的内容通过 JSON 文件管理，添加新文章/项目无需修改 HTML：

**`posts/posts.json`** — 文章索引
```json
{
    "slug": "文章文件名(不含.md)",
    "title": "文章标题",
    "date": "发布日期",
    "tags": ["分类标签"],
    "summary": "摘要",
    "cover": "封面类型(code/book/heart/gear/rocket/music)"
}
```

**`projects/projects.json`** — 项目索引
```json
{
    "slug": "项目文件夹名",
    "title": "项目名",
    "category": "项目分类",
    "status": "已完成|开发中",
    "date": "日期",
    "tags": ["技术栈标签"],
    "summary": "简介",
    "cover": "封面类型"
}
```

添加新内容只需两步：在对应文件夹创建 Markdown 文件 + 在 JSON 中添加一条索引记录。

## 视觉设计

### 暗黑太空主题

全局采用深色渐变背景（`#0a0a0a → #1a1429 → #2a1b3d → #0f172a`），模拟太空深邃感。前景元素使用毛玻璃材质（`backdrop-blur` + 半透明背景），在暗色底上形成清晰的层次。

### 蓝紫渐变色系

交互元素（按钮、链接、图标、分割线）统一使用 `from-blue-500 to-violet-500` 的蓝紫渐变，保持视觉一致性。hover 时增加发光阴影（`shadow-blue-500/25`）。

### 响应式设计

所有页面适配桌面端和移动端。使用 Tailwind 的响应式断点（`md:`、`lg:`），导航栏在手机上自动折叠为汉堡菜单，卡片布局从多列调整为单列，文字大小自适应。
