# 								网站说明

> 2026年6月 · 技术

## 缘起

一直想拥有一个属于自己的个人网站。作为一名电子信息工程专业的学生，与其用各种建站平台，不如亲手写一个（实际上是使用AI进行Vibe Coding）。

## 技术选型

这个网站的技术栈非常简单：

- **HTML5** — 页面结构
- **Tailwind CSS** — 通过 CDN 引入，无需构建工具，直接在 class 中写样式
- **JavaScript** — 原生的，没有用任何框架
- **GitHub Pages** — 免费托管，自动部署

选择这个组合的原因：**极简、够用、零依赖**。不需要 npm install，不需要 webpack 配置，写完直接 push 就能上线。

## 设计思路

### 暗黑太空主题

网站采用深色背景 + 蓝紫渐变的设计，灵感来自星空。整体色调偏暗但不沉闷，用 `backdrop-blur` 玻璃拟态效果增加层次感。

```css
.dark-space-bg {
  background: linear-gradient(to bottom, #0a0a0a 0%, #1a1429 40%, #2a1b3d 70%, #0f172a 100%);
}
```

### 中国古典诗词

Hero 区域加入了古典诗词随机轮播，目前收录了近百首李白、杜甫、苏轼等名家的诗词。这是整个网站最让我满意的细节——每次刷新都能看到不同的诗句。

### 写在最后

我会持续更新开发完善此网站，使之成为我回忆的一部分。
