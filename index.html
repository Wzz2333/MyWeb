关于我

个人介绍

我是一名前端开发者，热爱创造美观、实用的网站。专注于前端技术，熟悉 HTML、CSS、JavaScript 和现代前端框架。

技术栈

熟悉 HTML5、CSS3、JavaScript、Tailwind CSS、React 等前端技术，以及 Git、GitHub 等版本控制工具。

项目目标

本项目旨在创建一个高质量、现代、美观、响应式的静态网站，展示个人或项目信息，并通过 GitHub Pages 免费部署。

功能亮点

响应式设计

适配各种屏幕尺寸，从手机到桌面端都能完美显示。

深色模式

支持深色/浅色模式切换，保护眼睛，节省电量。

性能优化

轻量化设计，快速加载，提供流畅的用户体验。

GitHub Pages

免费部署，无需服务器，简单易用的静态网站托管服务。
