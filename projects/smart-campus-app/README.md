# 智慧校园助手

## 项目简介

为大学生校园生活打造的一站式服务平台，解决同学们在日常学习生活中遇到的各种痛点问题。

## 技术栈

- **前端**: Vue3 + TypeScript + Element Plus
- **后端**: Node.js + Express
- **数据库**: MySQL + Redis
- **部署**: Docker + Nginx

## 功能特性

### 课表查询
- 自动同步学校教务系统课表
- 支持周视图和日视图切换
- 上课提醒功能

### 成绩管理
- 可视化成绩分析图表
- GPA 自动计算
- 成绩趋势追踪

### 图书馆预约
- 在线选座预约
- 到馆签到
- 违规记录查询

### 校园资讯
- 通知公告推送
- 学术讲座信息
- 校园活动日历

## 项目亮点

1. **统一身份认证**: 对接学校统一身份认证系统，无需重复登录
2. **数据可视化**: 使用 ECharts 实现成绩、消费等数据的可视化展示
3. **响应式设计**: 完美适配手机、平板、电脑各种设备
4. **离线缓存**: 核心功能支持离线使用
