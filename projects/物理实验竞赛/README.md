## 一、设计原理与方法
### （一）设计原理
本实验依托**二维简谐振动理论**与**AI视觉测量技术**，实现小球二维振动轨迹自动观测与参数智能分析。整套系统将AI技术融入运动目标识别、轨迹预测、数据处理、结果分析全流程，借助计算机视觉完成非接触式自动测量，并通过本地AI模型实现实验状态判定与结果评价，有效提升实验效率、测量精度与可重复性。

根据胡克定律，弹簧回复力满足：
F=-kx


针对传统人工观测读数误差大、轨迹记录不完整等缺陷，本系统引入AI视觉识别技术，实现实验全过程自动化测量与分析。

### （二）设计方法
实验主体采用**弹簧-小铁球**振动结构，小球为振动核心，依靠上下弹簧提供回复力形成二维振动，配合水平拉杆产生水平位移；固定框架与导轨约束多余方向运动，保障实验稳定性。

测量环节通过摄像头实时采集振动视频，采用**模板匹配算法**定位非规则黑色小球，结合**Kalman滤波**抑制检测抖动，保证轨迹连续性。系统自动记录时间与坐标数据，利用SG滤波、FFT算法提取振动特征参数，最终由本地部署AI模型完成实验结果分析与评价，搭建智能化物理实验辅助体系。

## 二、实验仪器与装置
### 1. 二维简谐振动实验装置
装置为自主设计结构，由木质固定框架、弹簧系统、铁球质量块、水平拉杆、导轨组成。木质外框起到支撑、抗外界干扰作用，内部两组弹簧构成振动回复系统。

振动小球选用铁球外层包裹黑色布料制作，既保证质量分布均匀、振动平稳，又能与白色背景形成高对比度，便于视觉识别追踪。

实验操作：压缩弹簧后释放小球，同时拉动水平拉杆使系统产生水平位移，在弹簧弹力作用下形成标准二维简谐振动；导轨限制非目标方向运动，提升实验重复性与稳定性。

### 2. 图像采集系统
采用高清摄像头作为采集设备，**默认帧率60 fps，分辨率800×600**，高帧率可完整捕捉振动细节，避免高速运动造成轨迹缺失。

搭配白色背景板与补光灯，强化小球与背景的灰度差异，降低环境光线干扰；摄像头全程固定机位，杜绝视角偏移引入额外测量误差。

### 3. 运动目标追踪系统
基于**OpenCV计算机视觉库**实现非接触式测量：先人工框选目标区域生成小球模板，再通过模板匹配算法实时识别位置，该方案对不规则球体的适配性优于传统圆检测算法。

引入**Kalman滤波**对轨迹做预测与平滑处理，削弱摄像头噪声、检测误差带来的轨迹抖动，提升坐标数据的连续性与可靠性。

### 4. 数据处理与AI分析系统
整套软件基于**Python**开发，集成轨迹记录、数据运算、智能分析三大功能。系统自动采集小球二维坐标与时间序列数据，使用**Savitzky–Golay（SG）滤波**降噪、**快速傅里叶变换（FFT）**解析频率特征，提取周期、频率、振幅等物理参数。

系统接入**本地部署Ollama大模型**，针对轨迹图像、运动数据、频谱结果进行综合研判，自动完成振动规律判断、实验评价并输出优化建议，形成**实验观测—数据处理—智能分析**的闭环流程。

### 5. AI分析与软件系统
核心开发环境为Python，依托OpenCV实现目标检测与轨迹提取，借助NumPy、SciPy完成数值计算与频谱分析；本地Ollama模型负责数据与图像的智能解读。

区别于传统人工实验，本系统可全自动完成**目标识别→轨迹追踪→参数提取→结果分析**全流程，大幅提升实验自动化水平与数据分析效率。

## 三、数据测量与分析
1. **数据采集**
摄像头以60 fps帧率连续采集视频，追踪算法实时提取小球中心坐标，生成数据集 $(t,x,y)$，其中 $t$ 为时间，$x、y$ 分别为水平、竖直方向位移。相较于人工读数，该方式可连续记录完整运动轨迹，数据采集效率与精度显著提升。

2. **数据预处理**
实验初期受人为操作影响存在振动扰动，系统自动剔除前25%原始数据，仅保留稳定振动阶段数据；同时采用SG滤波对轨迹曲线平滑降噪，保留振动趋势的同时消除随机噪声。

3. **参数求解**
通过FFT算法识别振动主频，结合频率与周期的换算关系求解周期、频率；依据位移极值计算振幅，分别解析水平、竖直方向的运动特征，判定振动是否符合简谐振动规律。

4. **实验分析流程**
实验人员先结合轨迹图、时间-位移曲线、频谱图自主分析物理规律，再与AI模型分析结果交叉验证，自主研判振动状态、误差来源与参数变化规律，AI仅作为辅助评价、提供改进建议的工具。

系统可自动生成二维轨迹图、时间-位移曲线、平滑效果图，所有分析结果本地保存并可视化展示，实现**数据采集—参数分析—智能评价**一体化实验流程。

## 四、结论
本项目研制出一套基于AI视觉的二维简谐振动智能分析系统，实现振动轨迹自动观测、数据精准测量与结果智能分析。自主搭建的弹簧-铁球装置可稳定产生二维简谐振动，结合模板匹配与Kalman滤波算法，有效解决传统人工实验主观误差大、轨迹记录困难等问题。

数据处理模块通过SG滤波、FFT算法精准提取振幅、周期、频率等核心物理参数，完成振动规律定量研究；本地AI模型进一步提升数据解读效率，推动物理实验教学向智能化升级。

实测表明，本系统运行稳定、实时性强、可视化效果好、实验可重复性高。对比传统实验方案，在自动化程度、数据完整性、分析效率上具备明显优势，为AI技术在大学物理实验领域的落地提供可行案例，也为智能化实验教学建设提供参考思路。

## tracker_v4.py 技术架构（2026-05-23 最终状态）

### 三个窗口
1. **Ball Tracking**（800×600）：摄像头 + 绿色圈(Kalman位置) + 黄十字 + 状态栏
2. **Trajectory**（800×600）：绝对坐标轨迹(灰色原始+彩色Kalman样条)，网格50px
3. **Trajectory Zoom**（1000×750, 5×）：包围盒自适应居中，网格50px=10真实像素

### 追踪管线（主：模板匹配）
```
框选 → ball_template → matchTemplate(TM_CCOEFF_NORMED)
     → 置信度>0.45 → 更新Kalman → 记录轨迹
     → 低置信度 → Kalman预测推进
     → 长时间丢失 → Recovery(运动+暗色几何检测) → Global(纯暗色)
```

### 检测管线（备用：detect_ball）
```
灰度图 → GaussianBlur(5) → 固定阈值(dark_thresh, 默认70)
     → [可选] 背景差分 → 运动掩码 AND 暗色掩码
     → 开运算(3×3) → 闭运算(5×5)
     → FindContours → 几何评分(圆度+坚固度+范围+面积) → 最佳轮廓
     → 无结果? → 暴力回退: 取面积最大轮廓
```

### 状态机
```
STANDBY ←→ CLICK_INIT(拖拽框选) → TRACKING(模板匹配)
    ↑         ↑                        │
    └─────────┴──── SPACE/R键 ─────────┘
```

### Kalman 滤波器
- 4 状态（x, y, vx, vy），2 测量（x, y）
- processNoiseCov = 0.015
- 自适应 measurementNoiseCov：静止 8.0 → 快速 0.5
- 死区 2.0px（+/- 键调整 0.5-10.0px）

### 轨迹处理
- raw_points：最小位移 1.5px 过滤
- filtered_points：Kalman 滤波位置
- Catmull-Rom 样条插值（5 子步）平滑显示
- 放大窗口：包围盒中心 = (min+max)/2，自适应对准

### 按键操作
| 键 | 功能 |
|------|------|
| SPACE | 待机→框选 / 取消 |
| 鼠标拖拽 | 框选小球区域 → 锁定+5秒倒计时 → 自动开始追踪 |
| F | 生成正弦曲线图 + 保存CSV |
| Enter | 对F键生成的曲线进行SG平滑（给AI分析用） |
| Q | 退出 |
| R | 全部重置（轨迹+背景+模板+录制数据+正弦数据） |
| D | 调试窗口（Motion \| Dark \| Combined） |
| +/- | 调整死区大小 |
| [ / ] | 调整暗色阈值（30-120，步长5） |
