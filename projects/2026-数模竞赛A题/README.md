# 2026年辽宁省大学生数学建模竞赛 A题

> 辽宁冬季冰雪道路图像重构的智能增强

## 项目简介

针对辽宁省冬季冰雪雾天气下道路监控图像质量严重退化的问题，构建图像增强模型与综合评价体系。数据集包含 8,245 张辽宁冬季道路监控图像，按冰雪影响程度划分为清晰（clear）、轻度（light）、重度（heavy）三类场景。

### 三个子问题

- **问题一**：针对清晰场景道路图像，分析低光照、雾霾、反光等因素，建立图像增强模型提升视觉质量
- **问题二**：建立递进式图像增强模型，区分轻度薄雪与重度厚雪结冰场景，自适应恢复道路纹理与标线
- **问题三**：构建冰雪道路图像增强效果的综合评价指标体系，对增强结果进行定量与定性评价

## 技术栈

- **语言**: Python
- **图像处理**: OpenCV, NumPy, SciPy, scikit-image
- **深度学习**: PyTorch (Zero-DCE, ProgressiveEnhance)
- **数据分析**: Matplotlib, Pandas
- **环境**: RTX 4060 Laptop 8GB, CUDA

## 方法概述

### 传统方法
| 方法 | 原理 | 效果 |
|------|------|------|
| **CLAHE** | 自适应直方图均衡化，LAB空间分块处理 | ⭐ 最优，梯度提升 58-99% |
| 暗通道先验 | He et al. 2009 去雾算法 | 雾天效果好，重雪场景 SSIM 下降 |
| 自适应伽马 | 基于亮度自适应 γ 校正 | 保守（PSNR 最高但增强弱） |
| 同态滤波 | 频域分离光照与反射 | 重雪场景过曝 27%，不推荐 |
| MSRCR | 多尺度 Retinex 色彩恢复 | 丢失 14-18% 信息熵，不推荐 |

### 深度学习方法
- **Zero-DCE** (CVPR 2020)：零参考自监督增强，7层卷积约30K参数，自训练15轮
- **ProgressiveEnhance**：场景分类器（76%准确率）+ UNet编码解码器，分场景自适应增强

### 综合评价体系（问题三）
15个指标 × 4个维度：
- 全参考：PSNR、SSIM
- 无参考：信息熵、平均梯度、边缘密度、NIQE
- 任务驱动：边缘保持指数、车道线可检测性、局部方差
- 冰雪专用：过曝率、有效纹理比、雪覆盖估计、色彩饱和度

## 核心发现

- CLAHE 为综合最优方法：重度场景梯度从 42.4 → 96.4（**+98.6%**）
- PSNR 指标具有误导性：AutoGamma PSNR 最高（31.0）但梯度改善为负（-2.2%），证明多维评价体系的必要性
- 深度学习因训练轮次不足（15轮 vs 论文推荐200+）效果不及传统方法，但损失收敛趋势良好
